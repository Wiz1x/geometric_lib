
# Документация проекта

## Общее описание
Данный проект содержит функции для вычисления площади и периметра геометрических фигур: круга и квадрата.

## Описание функций

### Функции для круга:
1. **area(r)**  
   Принимает радиус круга (r) и возвращает его площадь.  
   **Пример вызова**:  
   `area(5)` — возвращает площадь круга с радиусом 5. Результат: 78.53981633974483.

2. **perimeter(r)**  
   Принимает радиус круга (r) и возвращает его периметр.  
   **Пример вызова**:  
   `perimeter(5)` — возвращает периметр круга с радиусом 5. Результат: 31.41592653589793.

### Функции для квадрата:
1. **area(a)**  
   Принимает сторону квадрата (a) и возвращает его площадь.  
   **Пример вызова**:  
   `area(4)` — возвращает площадь квадрата со стороной 4. Результат: 16.

2. **perimeter(a)**  
   Принимает сторону квадрата (a) и возвращает его периметр.  
   **Пример вызова**:  
   `perimeter(4)` — возвращает периметр квадрата со стороной 4. Результат: 16.
